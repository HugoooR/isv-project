Little project

The objective of this project is to create a web application that brings together several features on image transformation. You can navigate our different features using our menu. More information below.


Name Description
B&W Transformation to black and white of an image
Gray Transformation to shade of gray of an image
Resize Resize the size of an image (height, width)
Align_V Align vertically many images
Align_H Align horizontally many images
Fusion Merge of two image with an opacity and a position (x,y)
Animation Creation of a gif with many images

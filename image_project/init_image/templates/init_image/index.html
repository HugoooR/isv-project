Mini projet

L'objectif de ce projet est de réaliser une application web qui regroupe plusieurs fonctionnalités sur la transformation d'images. Vous pouvez naviguez sur nos différentes fonctionnalités à l'aide de notre menu. Plus d'information en dessous.


Name Description
B&W Transformation to black and white of an image
Gray Transformation to shade of gray of an image
Resize Resize the size of an image (height, width)
Align_V Align vertically two images
Align_H Align horizontally two images
Fusion Merge of two image with an opacity and a position
Animation Creation of a gif with many images
